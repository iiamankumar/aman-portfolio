# Setup Guide

## Setting Up Environment Variables
1. Create a `.env` file in the root of your project.
2. Add the necessary environment variables in the following format:
   ```
   VARIABLE_NAME=value
   ```

## Running Database Migrations
To run the database migrations, execute the following command:

```bash
npm run db:push
```

## Installing Dependencies
Make sure to install all the required dependencies by running:

```bash
npm install
```

## Running the Development Server or Building for Production
- To run the development server, use the following command:

```bash
npm run dev
```

- If you wish to build for production, execute:

```bash
npm run build
```
